# Apache Projects Overview

List of Open Source Apache projects grouped by functionality

Last updated on 26 Sep 2025

---

## Data Processing and Pipelines
- [Apache Airflow](https://projects.apache.org/project.html?airflow) — Workflow scheduler for data pipelines.  
- [Apache Beam](https://projects.apache.org/project.html?beam) — Unified batch/stream processing model with multiple runners.  
- [Apache Flink](https://projects.apache.org/project.html?flink) — Distributed stream and batch processing engine.  
- [Apache Spark](https://projects.apache.org/project.html?spark) — General-purpose distributed data processing engine.  
- [Apache Tez](https://projects.apache.org/project.html?tez) — DAG execution engine on Hadoop.  
- [Apache DolphinScheduler](https://projects.apache.org/project.html?dolphinscheduler) — Workflow scheduler for data engineering.  
- [Apache StreamPark](https://projects.apache.org/project.html?streampark) — DevOps platform for Flink/Spark streaming jobs.  
- [Apache Samza](https://projects.apache.org/project.html?samza) — Stream processing on Kafka/YARN.  
- [Apache Storm](https://projects.apache.org/project.html?storm) — Real-time stream processing.  
- [Apache Wayang (Incubating)](https://projects.apache.org/project.html?wayang) — Cross-platform data processing API/optimizer.  

## Storage, Databases, and Table Formats
- [Apache Cassandra](https://projects.apache.org/project.html?cassandra) — Distributed NoSQL wide-column database.  
- [Apache HBase](https://projects.apache.org/project.html?hbase) — Column-oriented database on HDFS.  
- [Apache Accumulo](https://projects.apache.org/project.html?accumulo) — Sorted key/value store with cell-level security.  
- [Apache Pinot](https://projects.apache.org/project.html?pinot) — Real-time OLAP datastore.  
- [Apache Druid](https://projects.apache.org/project.html?druid) — Real-time analytics database for time-series/events.  
- [Apache Kudu](https://projects.apache.org/project.html?kudu) — Columnar storage with fast analytics and updates.  
- [Apache Iceberg](https://projects.apache.org/project.html?iceberg) — High-performance table format for data lakes.  
- [Apache Hudi](https://projects.apache.org/project.html?hudi) — Transactional data lake storage for incremental processing.  
- [Apache Paimon](https://projects.apache.org/project.html?paimon) — Streaming data lake table store.  
- [Apache Parquet](https://projects.apache.org/project.html?parquet) — Columnar storage format for analytics.  
- [Apache ORC](https://projects.apache.org/project.html?orc) — Columnar storage format for analytics.  
- [Apache CarbonData](https://projects.apache.org/project.html?carbondata) — Indexed columnar storage for fast analytics.  
- [Apache IoTDB](https://projects.apache.org/project.html?iotdb) — Time-series database for IoT.  
- [Apache Doris](https://projects.apache.org/project.html?doris) — MPP analytical database.  
- [Apache AsterixDB](https://projects.apache.org/project.html?asterixdb) — Big data management for semi-structured data.  
- [Apache Phoenix](https://projects.apache.org/project.html?phoenix) — SQL layer over HBase.  
- [Apache Derby](https://projects.apache.org/project.html?derby) — Pure Java relational database.  
- [Apache Ozone](https://projects.apache.org/project.html?ozone) — Object store for Hadoop ecosystem.  
- [Apache TsFile](https://projects.apache.org/project.html?tsfile) — Columnar time-series file format.  
- [Apache Uniffle](https://projects.apache.org/project.html?uniffle) — Remote shuffle service for big data engines.  
- [Apache Celeborn](https://projects.apache.org/project.html?celeborn) — Remote shuffle service for compute engines.  
- [Apache Pegasus (Incubating)](https://projects.apache.org/project.html?pegasus) — Key-value storage service.  
- [Apache Kvrocks](https://projects.apache.org/project.html?kvrocks) — Redis-compatible KV database on RocksDB.  

## Messaging, Streaming, and Eventing
- [Apache Kafka](https://projects.apache.org/project.html?kafka) — Distributed event streaming platform.  
- [Apache Pulsar](https://projects.apache.org/project.html?pulsar) — Distributed pub/sub messaging and streaming.  
- [Apache RocketMQ](https://projects.apache.org/project.html?rocketmq) — Distributed messaging and streaming.  
- [Apache ActiveMQ](https://projects.apache.org/project.html?activemq) — JMS message broker.  
- [Apache Qpid](https://projects.apache.org/project.html?qpid) — AMQP messaging implementations.  
- [Apache EventMesh](https://projects.apache.org/project.html?eventmesh) — Serverless eventing infrastructure.  
- [Apache BifroMQ (Incubating)](https://projects.apache.org/project.html?bifromq) — Distributed MQTT broker.  
- [Apache Iggy (Incubating)](https://projects.apache.org/project.html?iggy) — Distributed commit-log/streaming system.  

## Data Integration, Ingestion, and Orchestration
- [Apache NiFi](https://projects.apache.org/project.html?nifi) — Dataflow automation and management.  
- [Apache Gobblin](https://projects.apache.org/project.html?gobblin) — Data integration and ingestion framework.  
- [Apache SeaTunnel](https://projects.apache.org/project.html?seatunnel) — Data integration/synchronization platform.  
- [Apache InLong](https://projects.apache.org/project.html?inlong) — One-stop massive data integration (message + ETL).  
- [Apache Sqoop (Attic)](https://projects.apache.org/project.html?sqoop) — Bulk transfer between Hadoop and RDBMS.  
- [Apache Crunch (Attic)](https://projects.apache.org/project.html?crunch) — APIs for MapReduce pipelines.  
- [Apache DataFu](https://projects.apache.org/project.html?datafu) — UDFs for data processing frameworks.  

## Query Engines and SQL
- [Apache Drill](https://projects.apache.org/project.html?drill) — Schema-free SQL engine for many datastores.  
- [Apache Impala](https://projects.apache.org/project.html?impala) — MPP SQL query engine for Hadoop storage.  
- [Apache Kyuubi](https://projects.apache.org/project.html?kyuubi) — Distributed multi-tenant SQL gateway.  
- [Apache Calcite](https://projects.apache.org/project.html?calcite) — Query planning/optimization framework and SQL parser.  
- [Apache Superset](https://projects.apache.org/project.html?superset) — Data exploration and BI visualization.  
- [Apache DataFusion](https://projects.apache.org/project.html?datafusion) — Rust-native query engine for Arrow data.  

## Governance, Metadata, and Catalog
- [Apache Atlas](https://projects.apache.org/project.html?atlas) — Data governance and metadata management.  
- [Apache Gravitino](https://projects.apache.org/project.html?gravitino) — Unified metadata/table service for data lakes.  
- [Apache Polaris (Incubating)](https://projects.apache.org/project.html?polaris) — Lake metadata and cross-engine catalog.  

## Serialization and Columnar Memory
- [Apache Arrow](https://projects.apache.org/project.html?arrow) — Columnar in-memory format and compute libs.  
- [Apache Avro](https://projects.apache.org/project.html?avro) — Data serialization with schemas and RPC.  

## ML/AI and Analytics
- [Apache SystemDS](https://projects.apache.org/project.html?systemds) — Scalable machine learning system.  
- [Apache MADlib](https://projects.apache.org/project.html?madlib) — In-DB machine learning on Postgres/Greenplum.  
- [Apache Mahout](https://projects.apache.org/project.html?mahout) — Scalable machine learning on distributed data.  
- [Apache TVM](https://projects.apache.org/project.html?tvm) — Machine learning compilation stack.  
- [Apache SINGA](https://projects.apache.org/project.html?singa) — Distributed deep learning.  
- [Apache OpenNLP](https://projects.apache.org/project.html?opennlp) — Natural language processing toolkit.  

## Search and Text
- [Apache Lucene](https://projects.apache.org/project.html?lucene) — Full-text search library.  
- [Apache Solr](https://projects.apache.org/project.html?solr) — Search platform built on Lucene.  
- [Apache OpenNLP](https://projects.apache.org/project.html?opennlp) — NLP toolkit.  
- [Apache Tika](https://projects.apache.org/project.html?tika) — Content detection and extraction.  

## Web and Application Servers
- [Apache HTTP Server](https://projects.apache.org/project.html?httpd) — Widely used web server.  
- [Apache Tomcat](https://projects.apache.org/project.html?tomcat) — Servlet/JSP container.  
- [Apache TomEE](https://projects.apache.org/project.html?tomee) — Java EE server built on Tomcat.  
- [Apache Geronimo](https://projects.apache.org/project.html?geronimo) — Java EE server components.  
- [Apache CXF](https://projects.apache.org/project.html?cxf) — Web services framework (JAX-WS/JAX-RS).  
- [Apache Axis2](https://projects.apache.org/project.html?axis2) — SOAP web services engine.  

## Security and Identity
- [Apache Shiro](https://projects.apache.org/project.html?shiro) — Application security (authc/authz).  
- [Apache Ranger](https://projects.apache.org/project.html?ranger) — Data security, authorization, governance.  
- [Apache Knox](https://projects.apache.org/project.html?knox) — Perimeter security for Hadoop services.  
- [Apache Santuario](https://projects.apache.org/project.html?santuario) — XML Signature/Encryption.  
- [Apache Kerby](https://projects.apache.org/project.html?kerby) — Kerberos libraries and KDC.  
- [Apache Fortress](https://projects.apache.org/project.html?fortress) — RBAC on LDAP.  

## Dev Tools, Build, and Release
- [Apache Maven](https://projects.apache.org/project.html?maven) — Build and dependency management.  
- [Apache Ant](https://projects.apache.org/project.html?ant) — Java build automation.  
- [Apache Ivy](https://projects.apache.org/project.html?ivy) — Dependency manager for Ant.  
- [Apache Gump](https://projects.apache.org/project.html?gump) — CI for ASF projects.  
- [Apache Yetus](https://projects.apache.org/project.html?yetus) — Release engineering/dev tooling.  
- [Apache Rat](https://projects.apache.org/project.html?rat) — Release audit tool (license headers).  
- [Apache Bigtop](https://projects.apache.org/project.html?bigtop) — Packaging/testing for big data stacks.  

## Logging and Observability
- [Apache Log4j](https://projects.apache.org/project.html?log4j) — Java logging framework.  
- [Apache log4net](https://projects.apache.org/project.html?log4net) — .NET logging.  
- [Apache log4cxx](https://projects.apache.org/project.html?log4cxx) — C++ logging.  
- [Apache Chainsaw](https://projects.apache.org/project.html?chainsaw) — Log viewer for Log4j.  
- [Apache SkyWalking](https://projects.apache.org/project.html?skywalking) — APM, tracing, metrics.  
- [Apache HertzBeat](https://projects.apache.org/project.html?hertzbeat) — Observability/monitoring platform.  

## Integration, ESB, and Microservices
- [Apache Camel](https://projects.apache.org/project.html?camel) — Integration framework and EIPs.  
- [Apache ServiceComb](https://projects.apache.org/project.html?servicecomb) — Microservice framework (Java).  
- [Apache Synapse](https://projects.apache.org/project.html?synapse) — ESB/mediation for SOA.  
- [Apache ServiceMix](https://projects.apache.org/project.html?servicemix) — ESB on OSGi/JBI.  

## Cloud, Infra, and Container Schedulers
- [Apache CloudStack](https://projects.apache.org/project.html?cloudstack) — IaaS cloud orchestration.  
- [Apache Mesos (Attic)](https://projects.apache.org/project.html?mesos) — Cluster resource manager.  
- [Apache YuniKorn](https://projects.apache.org/project.html?yunikorn) — Scheduler for container orchestrators.  
- [Apache Traffic Server](https://projects.apache.org/project.html?trafficserver) — High-performance caching proxy.  
- [Apache Traffic Control](https://projects.apache.org/project.html?trafficcontrol) — CDN control plane.  

## Big Data Ecosystem (Hadoop and Friends)
- [Apache Hadoop](https://projects.apache.org/project.html?hadoop) — HDFS, YARN, MapReduce core.  
- [Apache Hive](https://projects.apache.org/project.html?hive) — SQL data warehouse on Hadoop.  
- [Apache Oozie (Attic)](https://projects.apache.org/project.html?oozie) — Workflow scheduler for Hadoop.  
- [Apache Ambari](https://projects.apache.org/project.html?ambari) — Provision/manage Hadoop clusters.  
- [Apache Knox](https://projects.apache.org/project.html?knox) — Security gateway for Hadoop services.  

## Coordination and Configuration

- [Apache Curator](https://projects.apache.org/project.html?curator) — High-level ZooKeeper recipes.  
- [Apache Helix](https://projects.apache.org/project.html?helix) — Cluster management framework.  
- [Apache Ratis](https://projects.apache.org/project.html?ratis) — Raft consensus framework.

- [Apache Sling](https://projects.apache.org/project.html?sling) — Web framework over JCR.  
- [Apache JSPWiki](https://projects.apache.org/project.html?jspwiki) — Java wiki engine.  
- [Apache Roller](https://projects.apache.org/project.html?roller) — Blog platform.  
- [Apache OpenMeetings](https://projects.apache.org/project.html?openmeetings) — Web conferencing.  
- [Apache Guacamole](https://projects.apache.org/project.html?guacamole) — Clientless remote desktop gateway.  

## Office, Docs, and PDF

- [Apache PDFBox](https://projects.apache.org/project.html?pdfbox) — PDF creation and manipulation.  
- [Apache POI](https://projects.apache.org/project.html?poi) — Read/write Microsoft Office formats.  
- [Apache FOP](https://projects.apache.org/project.html?fop) — XSL-FO to PDF/print.

- [Apache Wicket](https://projects.apache.org/project.html?wicket) — Component-based Java web framework.

- [Apache MyFaces](https://projects.apache.org/project.html?myfaces) — JSF implementations/components.  
- [Apache Pekko](https://projects.apache.org/project.html?pekko) — Reactive distributed systems toolkit.

- [Apache SSHD](https://projects.apache.org/project.html?sshd) — SSH server/client in Java.  
- [Apache PLC4X](https://projects.apache.org/project.html?plc4x) — PLC protocol access libraries.  
- [Apache HttpComponents](https://projects.apache.org/project.html?httpcomponents) — HTTP client and core libraries.  
- [Apache Serf](https://projects.apache.org/project.html?serf) — High-performance HTTP client in C.

- [Apache Flagon](https://projects.apache.org/project.html?flagon) — UX analytics and telemetrics.
